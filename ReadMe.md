
## PDF Services :)
 

Usage
```text
usage: app_runner.py [-h] [-i  [...]] [-v] [-m] [-s] [-c]

optional arguments:

-h, --help            show this help message and exit
-i  [ ...], --input_pdf  [ ...]
                    Input PDF file or files
-v, --version         App version
-m, --merge           merge PDFs
-s, --split           split PDFs
-c, --convert         convert PDF to Doc
```
